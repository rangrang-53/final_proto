페르소나: 5060 중장년층
기능: 가전제품을 카메라로 찍으면 앱 내에서 인식하여 브랜드가 확인 가능하고, 작동법이 그림으로 나온다.
태스크:
1. 객체 감지
	- Grounding DINO: 제로샷 모델(재학습없이 새로운 물체도 인식되고 사용자가 입력한 prompt를 기반으로 찾게 되는 것)
		- 고정된 클래스 범위 내에서 빠르고 정확한 검출이 필요한 실시간 응용
	- YOLOv8: 많은 라벨 데이터로 학습해 여러 비전 태스크에 특화된 「지도 학습 기반 실시간 객체 감지 및 분류 모델 (자율주행에도 쓰임)
		- 학습하지 않은 새로운 클래스까지 탐지가 필요한 개방형 환경

=> 카메라 이미지는 우선 YOLOv8로 빠르게 처리하고, 특정 상황(새 브랜드나 미등록 제품 인식 필요 시)에서 Grounding DINO를 백엔드에 활용하여 추가 확인과 자동 라벨링 기능으로 정확도를 보완

2. 이미지 분류
3. 이미지 세분화
	- Segment Anything (SAM): 해당 bbox 기반 세분화

CLIP 이미지 분류 → SAM 이미지 세분화 → YOLOv8 tiny 로고 분류
